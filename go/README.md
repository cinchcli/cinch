# Generated Go bindings

This directory contains Go code generated from `crates/client-core/proto/cinch/v1/*.proto` via `buf generate`. **Do not edit by hand.**

To regenerate after a `.proto` change:

```bash
make generate
```

These bindings are the canonical source for the relay (`github.com/cinchcli/relay`), which vendors a synced copy under `relay/proto/cinch/v1/` and `relay/internal/cinchv1/`.
